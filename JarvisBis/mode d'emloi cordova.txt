INSTALLATION / MISE EN ROUTE CORDOVA

1) Installation en tout genre:
	- installer Node.js --> https://nodejs.org/en/download/

	- installer cordova -- > dans la console : npm install -g cordova

	- installer JDK --> http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
/!\ Attention tu dois être capable de retrouver l'endroit où est le fichier tu en auras besoin ensuite.

	- installer Android Studio --> https://developer.android.com/studio/index.html

2) Création d'une nouvelle variable d'environnement:
On vient de telecharger JDK mais il doit être reconnue comme une variable d'état, ce qui n'est actuellement pas le cas, on va donc s'en occuper.
	-> panneau de configuration
	-> Système et sécurité
	-> Système
	-> Paramètres système avancés (à gauche)
	-> Variable d'environnement... (onglet paramètre systeme avancés; en bas)
	->Nouvelle... (Variable système)
-> Nom de la variable: JAVA_HOME
-> Valeur de la variable: chemin menant au dossier jdk installé auparavant

3) Mise en route de Cordova:
Maintenant qu'on a bien tout mis en place, on va passer au démarage de cordova pour la première fois.

$cd path/SoFun  // on se place dans le dossier
$cordova platform add android   // on permet l'utilisation de android comme platform
$cordova platform add browser   // pareil avec le navigateur
$cordova build  //on build (la base)

4)Run le projet:
On passe ensuite au lancement du projet.

-Navigateur
$cordova run browser  // a faire dans le dossier du projet SoFun 
Le projet va se lancer dans le navigateur.
/!\ Attention tout modification faite dans un fichier quand le projet tourne sur le navigateur ne sera pas pris en compte même avec un refresh.
Il faut re lancer le projet pour voir les modifications (CTRL+c pour arreter le run puis $cordova run browser)

-Android
Tout d'abord, il va falloir activer le debuguage USB. C'est un paramètre developpeur, il faut donc activer ces paramètres si ce n'est pas déjà fait puis trouver l'option debuguage USB et l'activer.
L'android doit être relier en USB avec l'ordinateur.
$cordova run android
Et là si tout va bien une petit fenetre s'affiche sur le telephone demandant l'acces ou un truc du genre. On accepte bien évidemment.
Une fois que le script a fini on doit voir apparaitre sur le tel, l'application.
